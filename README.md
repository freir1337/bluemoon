## ðŸ› Troubleshooting

**Panel not showing on desktop?**
- Refresh your browser
- Check if extension is enabled in Extensions tab
- Ensure viewport width > 1000px

**Mobile FAB not appearing?**
- Check viewport width < 1000px
- Verify extension is enabled
- Clear browser cache

**Traits not applying to AI?**
- Ensure traits are saved
- Check that generation mode is "Together" or "Separate"
- Verify AI model supports prompt injection

**Settings not persisting?**
- Check browser's localStorage is enabled
- Clear cache and reload
- Try different browser if issues persist
